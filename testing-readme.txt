### Tester en une minute ###

1. lancer MainGraphical
2. taper "new" pour une nouvelle partie
3. entrer le nom
4. cliquer sur "Next day" trois ou quatre fois
5. Aller dans debug et reperer une transaction bien frauduleuse.
6. Cliquer sur Embassies, aller retrouver le fraudeur dans son pays et entrer son numero pour le mettre dans l'address book
7. Idem pour le compte frauduleux : retrouver le pays, faire b0 pour la première banque, numéro du compte
8. Cliquer sur "World Init" et entrer "2" pour une nouvelle dénonciation. Entrer 0,1 pour blamer les deux precedents
9. Si vous n'avez rien raté, et si je n'ai rien raté, l'écran devrait vous féliciter. Bravo !
